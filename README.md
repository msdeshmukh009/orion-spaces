# [Orion Spaces](https://orion-space.netlify.app/home)
  A social media for space geeks.
</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/msdeshmukh009/orion-spaces.git
$ cd orion-spaces
$ npm install
$ npm start
```

## **Features -**

- Create a post
  - Like  
  - Comments    
- Edit & delete post
- User feed   
  - Sort by latest, oldest & trending 
- User profile
  - Add profile picture
  - Bio
  - Portfolio URL
- Follow/Unfollow
- Explore feed
- Bookmark post
- Authentication
  - Sign-up
  - Login
  - Logout

## **Tech Stack -**

- React JS
- Redux toolkit + React-Redux
- Tailwind CSS
